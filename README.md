Airfryer Umrechner
Eine einfache Progressive Web App (PWA) zum Umrechnen von Backofenrezepten auf die Heißluftfritteuse — und umgekehrt.
Live: https://vegasjunkie.github.io/airfryer/

Funktion
Wer ein Rezept für den Backofen hat und es im Airfryer zubereiten möchte, muss Temperatur und Garzeit anpassen. 
Diese App übernimmt die Umrechnung automatisch.

Backofen → Airfryer

Ober-/Unterhitze: Temperatur −40 °C, Garzeit ×0,8
Umluft/Heißluft: Temperatur −20 °C, Garzeit ×0,8

Airfryer → Backofen

Temperatur +40 °C, Garzeit ÷0,8


Features

Beide Richtungen: Backofen → Airfryer und Airfryer → Backofen
Umschaltbar zwischen Ober-/Unterhitze und Umluft
Live-Berechnung ohne Button
Referenztabellen für Temperaturen und Garzeiten
Funktioniert auf Desktop und Handy
Installierbar als PWA (Homescreen-App auf Android/iOS)


Technisch
Reines HTML/CSS/JavaScript — kein Framework, keine Dependencies, kein Build-Prozess. Gehostet via GitHub Pages.


Alle Ergebnisse sind Richtwerte. Kurz vor Ende der Garzeit kontrollieren — Geräte variieren.
